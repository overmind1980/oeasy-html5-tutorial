# 图片元素

## 回忆

- 上次研究了 链接 配合内嵌对象的写法
	- a 的 target属性 负责指定打开窗口
	- iframe 的 name属性 负责定义窗口的名字
- 可以在网页中切换
	- 春夏秋冬
- 当然可以做出自己网站
	- 东南西北
	- 地水火风
	- MBTI
	- ...
- 这次就要求做一个自己的网站
- 快去试试吧！


### 总结 🤔

- 这次展示了各种好玩的网页
	- 都是基于a和iframe制作的


```
<a href="autumn/autumn.html" target="my_window">autumn</a>
<iframe name="my_window"></iframe>
```
