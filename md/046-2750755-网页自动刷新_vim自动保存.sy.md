### 自动保存

- 每次在空闲 3秒后自动保存文件缓存
	- 也就是.o.html.swp

```
set updatetime=3000 
```

- autocmd 自动命令会在
	- 光标 5 秒静止CursorHold（即没有键盘输入）后
	- 触发文件的保存
	- 使用 silent! 是为了避免在保存时出现消息
	- update 命令只在文件被修改后才保存
		- 与 :write 相比它不会不必要地写入未修改的文件

```
autocmd CursorHold * silent! update 
```
